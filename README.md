## セットアップ
事前にIntelliJ IDEA(Community Edition可)とgitのインストールをお願いします。

1. プロジェクトのclone
```bash
> git clone https://github.com/asoview/asoview-hands-on.git
```
2. IntelliJ IDEAにプロジェクトをImport
3. PositiveIntegerTest#createのテストを実行しパスすることを確認
